## 项目说明

本项目使用 **DDD**架构进行开发,服务端使用**SpringBoot + MyBatisPlus + ElasticSearch + Canal**来实现,数据库采用了**MySQL、Redis**

## 技术栈

### 后端

**SpringBoot + MyBatis + Redis + ElasticSearch + Canal**
